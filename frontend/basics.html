JavaScript basics



Here's some javascript basics like variable declaration, functions, loops etc.

As you can see, in JavaScript there are three different datatypes (var, let and const).
var (variable) is global if it's outside of a function. It's like "public" in Java.
let (variable) only available within its scope. If you use that outside of the scope, it will return "undefined".
const (variable) must be initialized and it maintain constant values. const values cannot be updated or re-declared.


Arrow functions allow us to write shorter function syntax. That's why arrow functions are used in the creation of stone API.
